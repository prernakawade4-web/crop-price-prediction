# crop-price-prediction
Smart Crop Price Prediction and Farmer Advisory System is an AI-based web application that predicts future crop prices using historical mandi and weather data. It employs Machine Learning models such as Random Forest and XGBoost, provides trend analysis, and helps farmers make informed selling decisions through a user-friendly dashboard.
# Smart Crop Price Prediction and Farmer Advisory System

## Overview
This project predicts future crop prices using Machine Learning techniques and historical mandi data.

## Features
- Crop Price Prediction
- Trend Analysis
- Farmer Advisory
- Weather Data Integration

## Technologies
- Python
- Flask
